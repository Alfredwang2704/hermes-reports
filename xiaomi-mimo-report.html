⚡ MiMo 核心技术亮点

🔀 混合注意力架构

滑动窗口注意力 (128 token) + 全局注意力按 5:1 比例交错，KV 缓存减少6 倍，同时保持长上下文性能。

🚄 多 Token 预测 (MTP)

轻量级 MTP 模块 (0.33B 参数/块)，原生集成训练和推理，推理速度提升 3 倍，加速 RL 训练 rollout。

🎓 多教师在线蒸馏 (MOPD)

token 级别密集指导，在线策略优化，避免奖励黑客问题，从多个专家模型学习。

🏗️ 大规模 Agent RL

10 万+ GitHub 真实任务，万级并发 Kubernetes 集群，70% 环境搭建成功率。

💡 评测总结

🎯 MiMo-V2-Flash 市场定位

  • Elo 评分 1393 — 位列 A 级优秀阵营，与 Grok 3 (1412)、MiniMax M2.5 (1404) 同级别
  • 推理能力强 — GPQA Diamond 83.7 分，AIME 2025 94.1 分，科学推理和数学能力优秀

  • 代码能力扎实 — SWE-Bench 73.4 分，超越 DeepSeek V3.2 (73.1 分)，具备优秀工程能力
  • 性价比高 — 作为开源模型，性能对标闭源旗舰，适合企业和开发者使用
